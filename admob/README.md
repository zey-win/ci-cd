# AdMob IDs

This folder is the handoff point for future game builds.

Primary AdMob account label: `AdMob Leopolda`.

For each app, prepare all supported AdMob values:

- Android app id
- Banner ad unit id
- Interstitial ad unit id
- Rewarded ad unit id
- Rewarded interstitial ad unit id
- Native ad unit id
- App open ad unit id

The `Build Android APK` workflow accepts these values as manual inputs:

- `admob_android_app_id`
- `admob_android_banner_id`
- `admob_android_interstitial_id`
- `admob_android_rewarded_id`
- `admob_android_rewarded_interstitial_id`
- `admob_android_native_id`
- `admob_android_app_open_id`

Current target apps:

- `slot_spot`
- `chicken-slots` (`Chicken World` in AdMob)
